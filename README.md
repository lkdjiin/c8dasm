# C8dasm

`c8dasm` is a utility to desassemble a Chip8 binary file

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'c8dasm'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install c8dasm

## Usage

    $ c8dasm chip8file
